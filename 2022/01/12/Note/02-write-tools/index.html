文字版

这期视频我们主要分享平时在 开发 、设计 、 写作 都会用到那些工具和网站。

视频中提到的工具和网站，在视频结尾，我会打包出来分享给大家，在公众号内回复 「工具」即可获取。

工具不在多，满足日常工作和学习的需求即可，同时我也开发了一个导航网站 ，这里面包含了很多工具，如果视频中提到的工具，不满足现在的需求， 也可以从这个网站上，找到你想要的工具，目前网站收录了 742 个网址。

正如你所见，我将网站分为了五个大类（前端开发、Android、设计导航、产品、运营），每个大的分类下面都有几个小的分类，如下图所示。

接下里进入这期视频的核心内容，先给大家展示一下我的 Chrome 主页，主要分为两个部分。

  • 顶部区域：是我们的收藏夹，收藏了常用的工具和网站。基本涵盖了工作和学习的需求
  • 中间区域：使用的是 Chrome 插件 Qlearly，以看板的形式，列出了每天都会用到的工具，以及需要去做的一些事情

Qlearly 它是一个书签管理工具，同时也是一个轻量级的 todo 和 笔记工具，它以列表的形式进行展示和管理，使用起来非常的方便。它分为 基础版本 和 高级版，基础版是完全免费的，但是没有同步的功能，高级版相比于基础版可以在多台设备上进行同步，在视频结尾，我会将 基础版和高级版 都导出来分享给大家。

  • 第一列：主要展示今天准备要做的事情，点击 Create Task 可以新建一个任务，点击 Create Note 可以新建一个笔记

  • 第二列：主要展示每天学习的网址
  • 第三列：主要展示每天常用的网址
  • 第四列：主要用来保存当天没有看完的网址，Qlearly 提供了一个非常好用的功能，点击 save 可以将我当前打开的网址全都保存下来，点击 open 可以将保存的网址全部打开

Qlearly 的作用就是帮助我们管理常用的网址，将那些不常用但是有非常重要的，可以放在顶部收藏夹中。因此我将收藏夹做了一个分类，这是我花了很长时间整理的，今天全部分享给大家。

设计工具和素材

设计工具

我们平时在写作的时候，需要去设计封面图，调整图片尺寸，对图片进行压缩等等。在设计工具里面可以找到这些工具，其中主要想介绍这几个工具。

Figma-设计工具

我主要用它来设计封面图：

  • 色板库： 我收集了一些好看的颜色，每次做封面图的时候，都会从里面取合适的颜色
  • 文章素材：包含了我目前所有文章的封面图，大家可以看到，封面图的尺寸和公众号封面图的尺寸大小是相同的。

在线修改图片工具

Online Image Tool 这个网站不仅免费，而且功能非常的强大，它可以 压缩任何图片格式 、转换图片格式 、 批量调整图片 、批量旋转图片 等等。

我们试试它的 压缩图片 功能，它的压缩效果非常的好，正如你所看到的 1.6MB 的图片压缩成了 560.4KB, 而且还可以对比原版和优化版。

代码图片生成工具

carbon 可以将你的代码生成漂亮的图片。

我们经常在网络上看到很多文章，将代码生成各种各样好看的图片，都是通过这个工具生成的，以下图片，是我其中一个开源项目 KtKit 封面图。

设计素材

主要包含了 阿里巴巴、Google 等国内外大厂提供的免费图标素材，以及各种矢量图，还有专门为开发者和设计师准备的素材。

发布平台

  • App 发布渠道：主要列了国内常用的 App 渠道
  • Maven 发布：用于将自己的开源项目发布在 Maven 上，方便其他人下载和使用
  • 文章发布：目前我的文章主要发布在掘金、51CTO、知乎、CSDN、简书
  • Vercel 网站托管平台：这个是我想重点介绍的，它是一个免费的网站托管平台，也是我目前用过最好的网站托管平台。目前我已经将我所有的网站都托管到这个平台上，主要有以下好处
    • 关联 github，只需要往 github 提交代码，它会自动获取最新的提交，然后自动部署
    • 提供了免费的域名，省去了申请域名的问题，如果有自己的域名，还可以做个域名解析到这个平台上
    • 提供了免费的 Https 证书，如果证书到期了，它会自动替换，完全不需要操心
  • Zapier：它是一个自动化工具，网络上对它的解释有很多，但是我把它比喻为 “连接”，它可以连接任意两个App，然后将这个两个App打通，使数据可以互相传递，举个例子，我在一个图片库的网站上(pinterest) 看到一个张非常好看的图片，点击收藏，可以自动同步到 Google 云盘、Dropbox, 又或者我在网络上看到一篇好文章，也可以自动同步到 Notion 中。

演示效果可以查看我的视频。目前 Zapier 已经支持 3000+ App 。

写作工具

本地写作工作工具

我主要使用的是 MWeb, 视频中的文字稿就是用它写的，它相比于其他的写作工具，有以下几个优点：

  • 它有很强大的文件夹管理功能
  • 内置图床上传功能：可以上传到 七牛云、阿里云、腾讯云、又拍云等等，这里有个建议，图片资源最好上传到自己的云空间里面，不要依托于第三方平台，目前我主要将图片上传到七牛云。

  • 内置发布服务：可以发布到印象笔记、语雀、为知笔记、少数派等等很多地方

  • 静态网站：这个功能对于不懂博客的同学是个福音，MWeb 不仅可以记笔记，写作，而且还是一个静态博客生成工具。

在线笔记

这里列了很多在线笔记工具，目前我主要在用 Notion、Obsidian、印象笔记、Cubox。关于这几个软件如何做联动，我会在下期的视频介绍。

Markdown 排版

这里列了很多在线排版工具，我主要将本地用 Markdown 写好的文章，通过这些排版工具，发布到微信公众号、知乎等等平台上，目前我主要在用 Typro 进行排版，它有几个好处：

  • 可以按照自己喜欢的风格，定制文章样式
  • 一键复制到 公众号 和 知乎

协同工具

在线云盘

  • 蓝奏云 和 闪电盘 这是非常好用的网盘工具，空间无限制、下载不限速，主要用于对外分享文件
  • Google 云盘 和 Dropbox 主要用来存储数据，而且它们还有另外一个好处，可以在多台设备上进行同步

转换工具

这里面包含了 文件格式的转换 、 正则表达式测试 、XML 格式转换 、二维码生成 等等，这里我主要想介绍正则表达式测试工具，这是一个非常好用的在线测试正则表达式的工具，不仅可以查看结果，而且还可以生成多种语言的代码。

网络工具

主要列了网络测速、安全检查等等网址

MAC 软件免费下载

如果你电脑是 MAC，那么这个网站必须收藏，这个网站包含了很多主流的软件，而且全部都是免费的。如果觉得软件好用，建议去购买正版。

下载工具

主要列了下载 Youtube 音乐和视频、下载 GooglePlay 上 App 的工具。

免费的 API

这里列了一些网上免费的 API，可以使用这些 API 开发一些开源项目。我之前的开源项目 神奇宝贝(PokemonGo)，就是用的这个 PokéAPI，目前在 github 上也有 1.3K star。

在线工具箱

在线工具箱 包含了很多工具比如 代码处理工具 、 站长工具 、 多媒体工具 、 便民工具，应该可以满足工作和生活的日常需求，如果在这里找不到，还有一个 Chorme 插件 FeHelper(前端助手) 里面也包含了很多工具。这两个工具结合起来一起使用应该可以满足大部分场景，在视频结尾，我会将这个插件导出来分享给大家。

画图工具

我们平时在写文档的时候，经常需要画思维导图、流程图、uml 图， 以及制作一些图表（流程图、饼状图等等），这里面的工具基本可以满足我们日常工作的需求。

  • processon：这是一个在线思维导图工具，我在整理收藏夹之前，就用 processon 做了一个思维导图，方便后面进行归纳
