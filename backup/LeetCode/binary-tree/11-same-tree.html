思路二：递归

复杂度分析：

  • 时间复杂度：O(min(m,n))，m 和 n 分别为两颗二叉树中的节点个数，对两个二叉树同时进行深度优先搜索，只要有一颗二叉树的节点为空，即停止循环，因此被访问到的节点数不会超过较小的二叉树的节点数
  • 空间复杂度：O(min(m,n))，m 和 n 分别为两颗二叉树中的节点个数，递归函数需要栈空间，而栈空间取决于递归的深度，因此空间复杂度等价于二叉树的高度，树的高度不会超过较小的二叉树的最大高度
